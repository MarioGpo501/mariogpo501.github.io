Pila

Una pila (stack) es una estructura de datos en la que los elementos se agregan y se eliminan desde el mismo extremo, llamado la “cima” o “tope”. Se describe como una estructura LIFO (Last In, First Out), es decir, el último elemento en entrar es el primero en salir.

Cola

Una cola (queue) es una estructura de datos en la que los elementos se agregan al final y se eliminan desde el principio. Se suele describir como una estructura FIFO (First In, First Out), es decir, el primer elemento en entrar es el primero en salir, como en una fila para esperar
